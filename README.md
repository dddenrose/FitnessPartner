Neo - Fitness Partner

主打兩項運動功能

1. HIIT 運動:可以選擇運動 ex: 登山者、深蹲
2. 超慢跑: 可以選擇步頻

# 流程

在 create-workout-plan 可以選擇要何種運動及細部設定。

1. HIIT -> 設定運動項目、組時間、組間休息時間

2. 超慢跑 -> 設定步頻，不限制時間

之後會進入 exercise 頁面，在這裡會看到兩種顯示模式

1. HIIT 模式 -> 檢視目前運動的項目，以及倒數的時間，下方功能列具有暫停、跳過當前運動、靜音倒數、完成此次運動、返回選擇運動的功能列

2. 超慢跑 -> 檢視節拍器、目前累計時間、下方功能列具有暫停、開關節拍器、完成此次運動、返回選擇運動的功能列

當完成運動後，有一個報表頁面能夠儲存當前使用者的跑步時間
